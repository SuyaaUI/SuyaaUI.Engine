# 舒雅UI引擎(SuyaaUI Engine)

舒雅UI引擎(SuyaaUI Engine) 是基于 .Net Standard 2.1 构建的跨平台UI引擎。

SuyaaUI使用Skia生成UI缓存图像，再配合Win32 MFC/Linux libX/OS Cocoa进行原生窗口生成与界面绘制。
